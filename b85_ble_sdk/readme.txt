To compile the source code with the eclipse project settings:
- Download the TELink TC BLE Single Connection SDK 
  https://github.com/telink-semi/tc_ble_single_sdk/releases
  https://doc.telink-semi.cn/doc/en/software/res/sdk/ble/b85m_ble_en/b85m_ble_single_connection_en
- Copy the content of the SDK folder here (containing "tl_common.h")
- The project was build with SDK version 3.4.2.4
